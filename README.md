# Dual Motor RPM Measurement & PWM Control using Hall Sensor (Arduino)

This project measures the RPM (Revolutions Per Minute) of **two DC motors** using **Hall effect sensors** and dynamically controls motor speed using **PWM signals** generated by Arduino.

The system uses **interrupt-based pulse counting** for accurate RPM calculation and maps the RPM values to PWM outputs in real time.
